The project sponsor - a major European manufactorer of agricultural macinery - has collected a unique dataset of foreign object events. Mutliple state of the art sensors are included (e.g. vibration, acoustics and mechanics) and measured with millisecond granularity.

The dataset contains more than a thousand foreign object events recorded under real-world field operating conditions.

The dataset is complemented by a corpus of data from normal field operating conditions without events.

In this project, participants will:

* build performant event detection algorithms, including core algorithm, feature extraction pipeline, pre-processing, using open technologies
* implement transparent performance measured to ensure the quality of the developed approaches, in terms of TPR, FPR, and advance detection time
* set up a model building workflow (using `sktime`) to rapidly test and evaluate algorithmic approaches for early event detection
* extend capabilities of `sktime` around the detection use case where necessary and useful, e.g., multivariate detection, semi-supervised detection, automated tuning, benchmarking detection algorithms

As stretch goals, participants in the project may:

* apply explainable AI or root cause analysis techniques to the pipeline, e.g., detecting which sensors or which part of the event was relevant for the detection
* participate in developing algorithms subject to the constraints of the embedded controller, using C and ONNX or similar

Join this project with a major European manufacturer of agricultural machinery and the German Center for Open Source AI, to build the next generation of agriculture machines!
